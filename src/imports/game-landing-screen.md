#mobile_app

Task: Design the public Game Landing screen shown when a user opens a shared game link.

Context:
Users may open a game through a shared link. Before starting gameplay, they should see basic information about the game.

------------------------------------------------

GAME INFORMATION

Game title

Example:
"Navidad 2026"

Metadata row:

Number of photos
Example: "6 fotos"

Creator name
Example: "Creado por Ana"

------------------------------------------------

PHOTO PREVIEW

Display the photos used in the game.

Thumbnail style:
- square
- rounded corners
- evenly spaced

------------------------------------------------

PRIMARY ACTION

Large button:

"Jugar"

------------------------------------------------

SECONDARY ACTION

Button:

"Ver leaderboard"

------------------------------------------------

OPTIONAL ACTION

Button:

"Crear tu propio juego"

------------------------------------------------

NOT FOUND HANDLING

If the game does not exist:

Redirect to existing screen:
"Juego no encontrado"

------------------------------------------------

CONSTRAINTS

- mobile-first layout
- keep the screen simple and inviting
- prioritize the "Jugar" button
- ensure photos visually communicate the game content
